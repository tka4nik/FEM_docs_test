void Console::Command::SetExecutionType ( CommandType  type)

Определеяет поток выполнения

Parameters
type- immediate или deferred

Команду можно выполнить сразу же в потоке Input, если ее выполнение не занимает много времени, или можно положить в очередь команд m_queue. Второй поток m_execution_thread (который сам запускается после создания Console) будет доставать команды из m_queue и выполнять их. На данный момент сделано так, что в m_execution_thread выполняется только Run. Все остальные команды выполняются сразу после ввода. Получается, что m_execution_thread полностью занят выполнением Run и в очередь вообще не смотрит. Для правильного поведения нужно, чтобы каждая команда, помеченная как deferred(или как-то по другому можно это назвать) создавала под себя отдельный поток (это не реализовано).
