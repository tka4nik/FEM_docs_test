24Класс Window унаследован от ErrorHandler, то есть в нем есть две очереди из строк ошибок и предупреждений
25Сейчас Window сохраняет только ошибки связанные с тесселяцией. PrepareData каждый шаг эти ошибки и предупреждения извлекает и выводит в консоль.
26В случае возникновения ошибки, пользователю предлагается продолжить моделирование или нет.
27Для реализации этого диалога в Console выставляется флаг m_dlg_require = true.
28Если это флаг выставлен функция Input не будет парсить введенную строку, а сохранит ее в m_dlg_input.
