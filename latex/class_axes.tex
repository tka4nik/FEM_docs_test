\doxysubsection{Detailed Description}
Класс для рисования осей Оси рисуются в два этапа. Сначала рисуются \char`\"{}тонкие оси\char`\"{} (положение которых фиксированно) Если \char`\"{}тонкие оси\char`\"{} вышли из области видимости, то рисуются линейки(rulers), прилипающие к соответствующим сторонам экрана. Функция \mbox{\hyperlink{class_axes_a8698ced15a031f58be6b24160abc2a8f}{Render()}} -\/ рисует тонкие оси(если это необходимо). Функция \mbox{\hyperlink{class_axes_a66fec7345407fab722c22c25d1ae75d3}{Post\+Render()}} -\/ линейки (если это необходимо). Каждый frame должен дополнительно вызываться Reset для актуализации информации о размере окна, смещении сцены и приближения.
