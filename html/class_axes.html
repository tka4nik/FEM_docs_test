Detailed Description

Класс для рисования осей Оси рисуются в два этапа. Сначала рисуются "тонкие оси" (положение которых фиксированно) Если "тонкие оси" вышли из области видимости, то рисуются линейки(rulers), прилипающие к соответствующим сторонам экрана. Функция Render() - рисует тонкие оси(если это необходимо). Функция PostRender() - линейки (если это необходимо). Каждый frame должен дополнительно вызываться Reset для актуализации информации о размере окна, смещении сцены и приближения.
